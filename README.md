# INOVA - Website Paket Usaha Laundry

Website promosi untuk bisnis paket usaha laundry yang menyediakan peralatan berkualitas tinggi.

## Fitur Utama

### 1. Halaman Utama (Landing Page)

- **Hero Section** - Penawaran bisnis dengan statistik 500+ mitra aktif, 98% kepuasan klien
- **Brand Intro** - Perkenalan brand INOVA
- **Paket Usaha** - 3 pilihan paket mesin laundry:
  - Paket SpeedQueen (3/4 mesin)
  - Paket LG Giant Max (3/4/Bisnis 4) - _Terlaris_
  - Paket Maytag Premium (3/4/5 mesin)
- **Produk** - Parfum, Setrika Uap, Deterjen
- **Kalkulator** - Kalkulator investasi bisnis
- **Mengapa INOVA** - Keunggulan bisnis
- **Proses Pemesanan** - Cara order
- **Testimoni** - Kata mitra
- **FAQ** - Pertanyaan umum
- **CTA Banner** - Ajakan menghubungi via WhatsApp

### 2. Fitur Tambahan

- **Bilingual** - Dukungan bahasa Indonesia & Inggris
- **Splash Screen** - Animasi pembuka
- **Responsif** - Tampilan optimal di mobile & desktop
- **Dark/Light Mode** - Tema gelap/terang
- **Integrasi WhatsApp** - Tombol langsung hubungi via WhatsApp

### 3. Sosial Media

- Instagram: @inovalaundrysolution
- TikTok: @inovalaundrysolution

## Teknologi

- Next.js 14
- React
- Tailwind CSS
- Framer Motion (animasi)
- Lucide Icons
