## Key Features by Section:

### Products Section (NEW) - Most Important
✨ **Visual Highlights:**
- Professional blue gradient backgrounds
- White frames with machine icons
- Large, bold pricing typography
- Feature lists with checkmarks
- Best Seller badge on Signature package
- Responsive: 3 columns (desktop), 1 column (mobile)
- Self-service in 4-column grid
- Decorative dots between subsections
- All prices in Rupiah (Rp Juta)

### Animations:
- Splash screen (2.5s animated intro)
- Scroll reveal on sections
- Staggered card animations
- Floating elements in Hero
- Smooth transitions on hover
- Loading dots in splash screen

### Responsive Design:
- Mobile-first approach (320px+)
- Tablet optimization (768px+)
- Desktop optimization (1024px+)
- All text scales appropriately
- Touch-friendly buttons (48px minimum)

### Dark Mode:
- System preference detection
- LocalStorage persistence
- All colors via CSS custom properties
- Smooth transitions

### Localization:
- Indonesian (id) - Default
- English (en)
- All text via i18next
- Dynamic formatting (dates, currency)
- Language selector in navbar
